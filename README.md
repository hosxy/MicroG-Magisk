# MicroG-Magisk

Your Rom have to support **Signature Spoofing**. Some custom ROMs are patched to support signature spoofing, but most ROMs will require a patch or a Xposed module.

The following ROMs have out-of-box support for signature spoofing:
+ AospExtended
+ ArrowOS
+ Havoc-OS
+ LineageOS for MicroG
+ OmniROM
+ etc ...

You can see more about **Signature Spoofing **[here](https://github.com/microg/android_packages_apps_GmsCore/wiki/Signature-Spoofing).

It include three apk in this module:

+ GmsCore.apk          v0.2.10.19420

+ Gsf.apk                    v0.1.0

+ GooglePlay.apk	    v17.6.19

You should open MicroG setting -> Self-Check and click `System grants signature spoonfing permission`, it will show a permission request dialog and click allow when you first use this module.

Please confirm all of box about Signature spoonfing support, Installed Package, Permissions granted have been checked.
